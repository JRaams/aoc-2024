# Advent of Code 2024

https://adventofcode.com/2024 by [Eric Wastl](http://was.tl/)

Solutions using TypeScript with Bun (v1.1.38)

## 1. Requirements

- Bun https://bun.sh/docs/installation

## 2. Running

`bun install`

Go to the folder of a specific day:

`cd src/01`

Run part A or B

`bun run a.ts`
